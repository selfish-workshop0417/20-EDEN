# 취향 큐레이션 저장 서비스

## 프로젝트 개요
좋은 제품을 발견했을 때 저장하고, 나중에 다시 볼 수 있는 개인 큐레이션 서비스

## 핵심 플로우
좋은 제품 발견 → 저장 → 링크/이미지/메모 기록 → 나중에 보기

## 주요 기능

### 1. 로그인
- 구글 소셜 로그인

### 2. 아이템 저장
- 제품 이름
- 링크
- 이미지
- 메모
- 태그

### 3. 리스트 보기
- 패션
- 신발
- 라이프스타일

### 4. 북마크
- 관심 아이템 저장

## 예시 아이템
- On Running 신발
- 아디다스 일본 제품
- 편집샵 제품

## 전체 사이트 구조

**총 6개 페이지 구조**
1. 랜딩
2. 홈/대시보드
3. 카테고리
4. 아이템 리스트
5. 아이템 상세
6. 아이템 저장/수정

## 기술 스택

- Backend/DB: Supabase
- 인증: Google OAuth
