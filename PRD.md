# taste-archive PRD

## 1. 제품 개요

**제품명:** taste-archive  
**한 줄 설명:** 위시리스트 저장부터 취향 기반 추천, 연예인 착용 아이템 대안 탐색까지 — 나만의 취향 아카이브

### 핵심 가치
- 흩어진 위시리스트를 한 곳에 모아 관리
- 저장된 아이템에서 취향을 학습해 새로운 상품 발견
- 비싸거나 품절된 연예인 착용 아이템의 현실적인 대안 제시

---

## 2. 문제 정의

| 사용자 불편 | 현재 해결 방식 | 한계 |
|---|---|---|
| 위시리스트가 인스타 저장, 네이버 즐겨찾기, 카카오톡 등에 분산 | 수동으로 링크 복사·저장 | 검색 불가, 맥락 없음 |
| 취향에 맞는 상품을 찾는 데 시간 낭비 | 무한 스크롤, 키워드 검색 | 피로도 높음, 발견 낮음 |
| 연예인 착용 아이템이 고가이거나 품절 | 직접 유사 상품 검색 | 시간 소요, 결과 불확실 |

---

## 3. 사용자

### 주 타겟
- **패션 관심 MZ (20–35세):** 연예인·인플루언서 스타일을 참고하고 합리적 대안을 찾는 사용자
- **온라인 쇼핑 헤비유저:** 여러 플랫폼에 걸쳐 위시리스트가 흩어져 있는 사용자

### 사용자 여정
```
발견 → 저장 → 탐색 → 대안 찾기 → 구매
```

---

## 4. 핵심 기능

### 4.1 위시리스트 저장
- URL 붙여넣기 또는 공유 시트로 상품 저장
- 상품명, 가격, 이미지, 브랜드 자동 파싱
- 태그·컬렉션으로 분류
- 가격 변동 알림

### 4.2 취향 기반 상품 추천
- 저장 아이템에서 스타일 벡터 생성 (색상, 카테고리, 브랜드, 가격대)
- 저장·조회·클릭 행동으로 선호 지속 업데이트
- 피드 형태로 개인화 추천 노출
- "이것과 비슷한 상품" 아이템 레벨 추천

### 4.3 연예인 착용 아이템 대안 제시

#### 기능 개요
연예인 착장 이미지 또는 이름+키워드를 입력하면, 해당 아이템을 인식하고 유사한 스타일의 대안 상품을 가격대별로 추천한다.

#### 사용자 시나리오

**시나리오 A — 이미지 검색**
1. 사용자가 인스타그램·유튜브에서 연예인 착장 이미지를 캡처
2. taste-archive 앱 내 "연예인 탐색" 탭 진입
3. 이미지 업로드 또는 URL 붙여넣기
4. AI가 착장 내 아이템 분석 → 카테고리별 태그 표시 (상의, 하의, 신발, 액세서리 등)
5. 각 아이템의 원본 정보(브랜드, 추정 가격, 구매 링크) 확인
6. "대안 보기" 탭에서 유사 스타일 상품 리스트 탐색 (가격대 슬라이더로 필터)
7. 마음에 드는 대안 상품 위시리스트에 저장

**시나리오 B — 이름 검색**
1. 사용자가 "아이유 오늘 공항 패션" 텍스트 검색
2. 크롤링된 최신 착장 데이터에서 해당 아이템 목록 노출
3. 이후 시나리오 A의 5–7단계와 동일

**시나리오 C — 실패 케이스 처리**
- 이미지 내 아이템 인식 실패 시: "직접 카테고리 선택" 수동 탐색으로 전환
- 원본 상품 정보 없을 시: 유사 스타일 추천만 제공, 원본 정보는 "정보 없음" 표시

#### 데이터 구조

```typescript
// 연예인 엔티티
interface Celebrity {
  id: string;
  name: string;               // "아이유"
  aliases: string[];          // ["IU", "이지은"]
  profileImageUrl: string;
  instagramHandle?: string;
}

// 착장 피드 아이템
interface CelebrityOutfit {
  id: string;
  celebrityId: string;
  sourceImageUrl: string;     // 원본 이미지 URL
  sourceUrl?: string;         // 출처 (인스타, 기사 등)
  occasion?: string;          // "공항", "방송", "일상" 등
  detectedAt: Date;
  items: OutfitItem[];
}

// 착장 내 개별 아이템
interface OutfitItem {
  id: string;
  outfitId: string;
  category: "top" | "bottom" | "outer" | "shoes" | "bag" | "accessory";
  boundingBox?: BoundingBox;  // 이미지 내 위치
  originalProduct?: {
    brandName: string;
    productName: string;
    estimatedPrice: number;
    currency: "KRW" | "USD";
    purchaseUrl?: string;
    isAvailable: boolean;
  };
  styleVector: number[];      // 임베딩 벡터 (추천에 활용)
  alternatives: AlternativeProduct[];
}

// 대안 상품
interface AlternativeProduct {
  id: string;
  outfitItemId: string;
  productId: string;          // 위시리스트 Product 참조
  similarityScore: number;    // 0–1, 스타일 유사도
  priceRange: "budget" | "mid" | "premium";  // 예산별 분류
  rankOrder: number;
}

// 이미지 내 바운딩 박스
interface BoundingBox {
  x: number;    // 좌상단 x (0–1 정규화)
  y: number;    // 좌상단 y
  width: number;
  height: number;
}
```

#### 추천 로직 개요
1. **스타일 유사도:** 아이템 이미지 임베딩 코사인 유사도 기반
2. **가격대 필터:** budget(~5만원), mid(5–20만원), premium(20만원+) 3단계
3. **재고 우선:** 품절 상품은 리스트 하단으로 내림
4. **취향 보정:** 사용자 위시리스트 벡터와 교차하여 개인화 적용

## 6. 비기능 요구사항

- **URL 파싱 지원 쇼핑몰:** 무신사, 29CM, 쿠팡, 네이버쇼핑, 지그재그, 에이블리 (v1 기준)

- **플랫폼:** iOS + Android (웹 뷰 포함), 웹 대응은 v2
