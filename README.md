# 취향 큐레이션 저장 서비스

좋은 제품을 발견했을 때 저장하고, 나중에 다시 볼 수 있는 개인 취향 큐레이션 서비스

## 프로젝트 정보

- **유형**: 웹 서비스 (SaaS-lite)
- **타겟 사용자**: 패션·라이프스타일에 관심 있는 개인
- **출시 목표**: 1개월 이내

## 핵심 플로우

좋은 제품 발견 → 저장 → 링크/이미지/메모 기록 → 나중에 보기

## 팀 구성

| 파트 | 역할 |
|------|------|
| planning | 서비스 기획 및 PRD 관리 |
| design | UI/UX 디자인 |
| frontend | 화면 개발 (React) |
| backend | 서버 및 DB 개발 (Supabase) |
| cx | 사용자 경험 및 콘텐츠 |

## 기술 스택

- **Frontend**: React
- **Backend/DB**: Supabase
- **인증**: Google OAuth

## 협업 방식

1. `main` 브랜치: 최종 완성본
2. `feature/파트명` 브랜치: 각 파트별 개발
3. PR → 코드 리뷰 → 머지
